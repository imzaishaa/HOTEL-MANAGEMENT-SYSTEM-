# HOTEL-MANAGEMENT-SYSTEM-
Hotel Management System is a comprehensive solution that uses SQL for backend data management and Microsoft Access as a user-friendly frontend interface. It handles room bookings, customer information, staff details, and reporting, streamlining hotel operations. 

## Overview
This Hotel Management System is designed to help hotel staff manage daily operations efficiently. It combines a robust SQL database backend with a Microsoft Access frontend to provide a seamless and user-friendly experience for managing rooms, bookings, customers, and staff information.

## Features
- **Room Management:** Track room types, availability, and pricing.
- **Booking Management:** Create and manage guest reservations.
- **Customer Management:** Store guest details and booking history.
- **Staff Management:** Maintain employee records and roles.
- **Reporting:** Generate reports on bookings, occupancy, and revenue.
- **User Interface:** Microsoft Access frontend offers easy navigation and data entry without requiring SQL knowledge.

## Technologies Used
- **Backend:** SQL (MySQL / SQL Server / specify your choice)
- **Frontend:** Microsoft Access (forms, queries, reports)
- **Connection:** ODBC or linked tables to connect Access to the SQL database

## Installation
1. Import the provided SQL scripts to create the database schema and populate sample data.
2. Open the Microsoft Access frontend file (`HotelManagement.accdb`).
3. Configure the ODBC connection or link tables to connect the Access frontend to the SQL backend.
4. Use the Access forms to manage hotel operations.

## Usage
- Use the **Room Management** form to add, update, or delete rooms.
- Use the **Booking Management** form to make new reservations and check booking status.
- Manage customer information using the **Customer Management** form.
- Staff details can be viewed and edited in the **Staff Management** form.
- Generate and view various reports to monitor hotel performance.
